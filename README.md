# 勒克莱尔模拟器 下载说明

按设备下载对应压缩包：

- Windows：`simulator_16_light_win.zip`
- macOS：`simulator_16_light_mac.zip`
- 移动端：`simulator_16_mobile.zip`

## 使用方法

1. 下载对应压缩包
2. 解压
3. 保持 `assets` 文件夹和 html 文件在同一目录
4. 打开对应的 html 文件即可

## 对应文件

- Windows 包内主文件：`simulator_16_light.html`
- macOS 包内主文件：`simulator_16_light.html`
- 移动端包内主文件：`simulator_16_mobile.html`
